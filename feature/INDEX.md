## How to Add a New Feature

1. Create a new `.md` file in this `/feature/` folder describing the feature.
2. Add a row to the table above.
3. Create a corresponding plan file in `/plan/` if architecture decisions are involved.
4. Implement according to the extension guide in `/plan/ARCHITECTURE.md`.
